# homebrew-tools

This repository represents [Homebrew Tap][web:homebrew/tap] containing software from _norcane_ project. You can install selected package using `brew install norcane/tools/<FORMULA>`.

## 1. Available Formulas

- [Headroom][github:headroom] - Manager for license headers in source code files
  ```
  $ brew install norcane/tools/headroom
  ```

[github:headroom]: https://github.com/vaclavsvejcar/headroom
[web:homebrew/tap]: https://docs.brew.sh/Taps
